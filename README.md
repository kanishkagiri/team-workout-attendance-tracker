# Team Workout & Attendance Tracker

A full-stack web application designed to help coaches manage team members, schedule training sessions, and track attendance efficiently.


## Project Overview

The Team Workout & Attendance Tracker enables:

- Managing team members
- Scheduling training sessions
- Recording attendance (Present / Absent)
- Viewing attendance summaries
- Generating weekly training reports

## Features

### 1️ Team Management
- Add new team members
- Update member details
- Delete members
- View all registered members

### 2️ Training Sessions
- Create training sessions
- Set session date and description
- View all sessions
- Update or delete sessions

### 3️ Attendance Tracking
- Mark attendance (Present / Absent)
- View attendance per session
- Update attendance records

### 4️ Reports
- View attendance summary per member
- Generate weekly training reports
